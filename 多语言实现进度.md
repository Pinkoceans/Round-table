## 📝 改造方法

### 示例：修改 SettingsPanel.jsx

**修改前：**
```jsx
import React from 'react';

function SettingsPanel() {
  return (
    <div>
      <h2>AI 配置管理</h2>
      <input placeholder="例如：我的 AI 配置" />
      <button>保存</button>
    </div>
  );
}
```

**修改后：**
```jsx
import React from 'react';
import { useForumStore } from '../store';
import { t } from '../i18n';

function SettingsPanel() {
  const { language } = useForumStore();
  
  return (
    <div>
      <h2>{t('settings.title', language)}</h2>
      <input placeholder={t('settings.placeholder.name', language)} />
      <button>{t('action.save', language)}</button>
    </div>
  );
}
```

### 步骤：
1. 导入 `useForumStore` 和 `t` 函数
2. 从 store 中获取当前语言：`const { language } = useForumStore();`
3. 将所有硬编码的中文文本替换为 `{t('key', language)}`
4. 在 `i18n.js` 中添加缺失的翻译键

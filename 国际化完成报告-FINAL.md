## 🔒 严格单语言模式

### 实现方式
所有组件使用统一的翻译模式：
```jsx
import { useForumStore } from '../store';
import { t } from '../i18n';

function Component() {
  const { language } = useForumStore();
  
  return (
    <div>
      <h1>{t('key', language)}</h1>
      {/* 只显示当前语言的文本 */}
    </div>
  );
}
```

### 语言切换效果
- 🇨🇳 **切换到中文** → 界面 100% 中文，无英文/日文
- 🇬🇧 **切换到 English** → 界面 100% English，无中文/日文
- 🇯🇵 **切换到 日本語** → 界面 100% 日本語，无中文/英文

### 零容忍政策
❌ 不允许任何硬编码文本  
❌ 不允许 fallback 到其他语言  
❌ 不允许混合语言显示

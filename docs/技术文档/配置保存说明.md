## 📊 数据存储位置

### 浏览器本地存储（localStorage）
数据保存在你的浏览器中，具体位置：

- **Chrome**: `设置` → `隐私和安全` → `网站设置` → `本地存储`
- **Safari**: `偏好设置` → `隐私` → `管理网站数据`
- **Firefox**: `选项` → `隐私与安全` → `Cookie 和网站数据`

### 存储键名：
- `yuanzhuo_topics` - 所有话题
- `yuanzhuo_current_topic` - 当前话题
- `yuanzhuo_active_characters` - 已选择的角色
- `yuanzhuo_api_configs` - AI 配置

---

## 🔒 隐私和安全

### 数据存储在哪里？
- ✅ 数据只保存在**你的浏览器本地**
- ✅ **不会上传**到任何服务器
- ✅ **不会分享**给第三方
- ✅ 只有你能访问这些数据

### 清除数据：
如果你想清除所有保存的数据：

**方法 1：清除浏览器数据**
```
浏览器设置 → 清除浏览数据 → 选择"本地存储" → 清除
```

**方法 2：使用开发者工具**
1. 按 F12 打开开发者工具
2. 进入 `Application` → `Local Storage`
3. 删除所有 `yuanzhuo_` 开头的键
4. 刷新页面

---

## ⚠️ 注意事项

### 1. 浏览器特定
- 数据保存在**当前浏览器**中
- 在不同浏览器（如 Chrome → Safari）访问，数据不会同步
- 在不同设备访问，数据不会同步

### 2. 隐私模式
- 如果使用**隐私/无痕模式**，关闭浏览器后数据会丢失
- 建议使用**普通模式**以获得持久化保存

### 3. 数据备份
由于数据只保存在本地，建议：
- 重要聊天记录可以手动复制保存
- 定期截图保存重要讨论

## 🛠️ 故障排除

### 问题：刷新后数据丢失
**可能原因**：
- 使用了隐私/无痕模式
- 清除了浏览器缓存
- 浏览器设置阻止了本地存储

**解决方法**：
- 使用普通模式（非隐私模式）
- 检查浏览器设置，允许本地存储
- 不要清除浏览器缓存

### 问题：数据没有保存
**可能原因**：
- 页面加载失败
- JavaScript 错误
- 浏览器兼容性问题

**解决方法**：
- 打开浏览器控制台（F12）查看错误
- 确保使用的是现代浏览器（Chrome、Safari、Firefox 等）
- 刷新页面重试

## 📝 技术实现

### 使用的技术：
- **Zustand** - 状态管理库
- **localStorage** - 浏览器本地存储
- **JSON** - 数据序列化

### 核心函数：
```javascript
// 从本地存储加载
loadFromStorage(key, defaultValue)

// 保存到本地存储
saveToStorage(key, value)
```

### 自动保存流程：
```
用户操作 → Zustand 状态更新 → saveToStorage → localStorage
```

### 自动加载流程：
```
页面加载 → createStore → loadFromStorage → 恢复状态
```

## 📚 内容规划

### 基础技巧（4 个）
1. 🎯 明确具体的问题
2. 📚 提供充分的背景信息
3. 📋 指定回答格式
4. 🔄 迭代式提问

### 高级技巧（4 个）
1. 🎭 角色扮演法
2. 🧩 分步思考引导
3. 🔍 多角度分析
4. ⚖️ 约束条件设定

### 场景示例（4 个）
1. 📖 学习辅导（3 个示例）
2. 💻 代码编程（3 个示例）
3. ✍️ 写作创作（3 个示例）
4. 📊 数据分析（2 个示例）

### 常见误区（4 个）
1. ❌ 问题太模糊
2. ❌ 期望一次完美
3. ❌ 不提供上下文
4. ❌ 不问跟进问题

### 快速技巧（6 条）
- 明确：问题越具体越好
- 背景：提供相关信息
- 格式：指定回答方式
- 迭代：多轮对话深化
- 角色：设定专业身份
- 步骤：要求展示过程

### 问题模板（16 个）
- 学习类：4 个模板
- 编程类：4 个模板
- 写作类：4 个模板
- 分析类：4 个模板

## 📋 配置说明

### 立即可用（测试环境）
- **手机号登录**：使用测试验证码 `123456`

### 需要配置（生产环境）

#### 1. GitHub 登录
```bash
配置步骤：
1. 访问 https://github.com/settings/developers
2. 创建 OAuth App
3. 配置回调 URL
4. 获取 Client ID 和 Secret
5. 在 Netlify Identity 配置
```

#### 2. Google 登录
```bash
配置步骤：
1. 访问 https://console.developers.google.com/
2. 创建 OAuth 2.0 客户端
3. 配置回调 URL
4. 获取 Client ID 和 Secret
5. 在 Netlify Identity 配置
```

#### 3. 微信登录（使用 Authing）
```bash
配置步骤：
1. 访问 https://authing.cn/
2. 注册并创建用户池
3. 启用微信登录
4. 配置微信开放平台
5. 在应用中使用 Authing SDK
```

#### 4. 短信服务（生产环境）
```bash
阿里云短信配置：
1. 访问 https://dysms.console.aliyun.com/
2. 创建短信签名和模板
3. 获取 AccessKey
4. 在 Netlify 配置环境变量
```

## 🚀 使用指南

### 本地开发
```bash
# 启动开发服务器
npm run dev

# 访问 http://localhost:5173/
# 点击登录按钮
# 选择手机号登录
# 输入任意 11 位手机号
# 验证码输入：123456
```

### 部署到 Netlify
```bash
# 使用部署脚本
./deploy.sh

# 或手动部署
git add .
git commit -m "Update login features"
git push origin main
```

### 配置 OAuth
1. 访问 Netlify 控制台
2. 进入 Site settings → Identity
3. 启用 Identity 服务
4. 配置 External providers
5. 添加 GitHub、Google 等

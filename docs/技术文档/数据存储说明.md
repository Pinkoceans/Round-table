# 📦 数据存储说明

## ✅ 数据存储机制

### 数据保存在哪里？

你的所有数据都保存在**浏览器的本地存储（localStorage）**中，位置：
- **浏览器** → **开发者工具** → **Application** → **Local Storage** → **http://localhost:5173**

### 保存的内容：

1. **API 配置** (`yuanzhuo_api_configs`)
   - 你添加的所有 AI 服务商配置
   - API 密钥、模型选择、温度设置等
   - ✅ **不会因为代码修改而丢失**

2. **已选择的角色** (`yuanzhuo_active_characters`)
   - 你当前选择的角色列表
   - ✅ **刷新页面后仍然存在**

3. **话题和聊天记录** (`yuanzhuo_topics`)
   - 创建的所有话题
   - 完整的聊天记录
   - ✅ **永久保存，除非手动删除**

4. **对话设置** (`yuanzhuo_dialog_settings`)
   - 发言次数限制
   - 间隔时间
   - 互动模式开关
   - ✅ **自动保存**

5. **当前话题** (`yuanzhuo_current_topic`)
   - 当前正在查看的话题 ID
   - ✅ **下次打开时自动恢复**

## 🔍 如何查看保存的数据

### 方法 1：使用浏览器开发者工具

#### Chrome / Edge:
1. 按 `F12` 打开开发者工具
2. 点击 **Application** 标签
3. 左侧选择 **Local Storage** → **http://localhost:5173**
4. 右侧可以看到所有保存的数据

#### Safari:
1. 按 `Cmd + Option + I` 打开开发者工具
2. 点击 **Storage** 标签
3. 展开 **Local Storage** → **http://localhost:5173**
4. 查看保存的数据

#### Firefox:
1. 按 `F12` 打开开发者工具
2. 点击 **Storage** 标签
3. 展开 **Local Storage** → **http://localhost:5173**
4. 查看保存的数据

## ❓ 常见疑问

### Q1: 为什么每次修改代码后，数据好像丢失了？

**A**: 数据**没有丢失**！只是：
- 修改代码后，浏览器会自动刷新
- 刷新需要时间（1-3 秒）
- 刷新完成后，数据会自动加载回来

**验证方法**：
1. 添加一个 AI 配置
2. 修改代码（任何地方）
3. 等待浏览器自动刷新完成
4. 打开设置面板，查看 AI 配置 → **它还在！**

### Q2: 数据会永久保存吗？

**A**: 是的，但有以下情况会丢失：
- ✅ **代码修改** → 数据不会丢失
- ✅ **刷新页面** → 数据不会丢失
- ✅ **关闭浏览器** → 数据不会丢失
- ❌ **清除浏览器缓存** → 数据会丢失
- ❌ **使用隐私模式** → 关闭后数据会丢失

### Q3: 如何在不同浏览器间同步数据？

**A**: localStorage 是**浏览器特定**的：
- Chrome 的数据不会同步到 Safari
- 手机和电脑的数据不互通
- 如需同步，需要手动导出导入（未来功能）

## 🛠️ 数据管理

### 查看数据内容

在浏览器控制台中执行：

```javascript
// 查看所有 API 配置
console.log(JSON.parse(localStorage.getItem('yuanzhuo_api_configs')));

// 查看已选择的角色
console.log(JSON.parse(localStorage.getItem('yuanzhuo_active_characters')));

// 查看所有话题
console.log(JSON.parse(localStorage.getItem('yuanzhuo_topics')));

// 查看对话设置
console.log(JSON.parse(localStorage.getItem('yuanzhuo_dialog_settings')));
```

### 导出数据（备份）

在浏览器控制台中执行：

```javascript
// 导出所有数据
const backup = {
  apiConfigs: localStorage.getItem('yuanzhuo_api_configs'),
  activeCharacters: localStorage.getItem('yuanzhuo_active_characters'),
  topics: localStorage.getItem('yuanzhuo_topics'),
  dialogSettings: localStorage.getItem('yuanzhuo_dialog_settings'),
};

console.log('备份数据：');
console.log(JSON.stringify(backup, null, 2));
```

### 导入数据（恢复）

⚠️ **谨慎操作**：

```javascript
// 从备份恢复
const backup = {
  apiConfigs: '[你的备份数据]',
  // ... 其他数据
};

Object.keys(backup).forEach(key => {
  localStorage.setItem(`yuanzhuo_${key}`, backup[key]);
});

location.reload(); // 刷新页面
```

### 清除数据

⚠️ **危险操作，会删除所有数据**：

```javascript
// 清除所有圆桌数据
localStorage.removeItem('yuanzhuo_api_configs');
localStorage.removeItem('yuanzhuo_active_characters');
localStorage.removeItem('yuanzhuo_topics');
localStorage.removeItem('yuanzhuo_current_topic');
localStorage.removeItem('yuanzhuo_dialog_settings');

location.reload(); // 刷新页面
```

或者在开发者工具中：
1. 打开 **Application** → **Local Storage**
2. 右键点击 `yuanzhuo_` 开头的键
3. 选择 **Delete**

- 📱 数据不会自动同步
- 📱 可以手动复制配置

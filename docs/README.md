## 📖 技术文档 (`/docs/技术文档`)

- `AI 对话增强功能说明.md` - AI 对话功能详细说明

- `PROJECT_GUIDELINES.md` - 项目开发指南
- `CLEANUP_SUMMARY.md` - 项目清理总结

### 用途：
- 了解项目功能
- 学习技术实现
- 开发参考
- 功能说明

## 🔧 配置指南 (`/docs/配置指南`)

**⚠️ 重要提示**：此目录包含需要用户手动配置的操作指南，**完成配置后可以删除整个目录**。

### 包含文件：
- `邮箱配置指南.md` - 配置建议接收邮箱
- `快速设置.md` - 快速配置指南
- `Netlify 部署教程.md` - Netlify 部署步骤
- `永久部署方案.md` - 永久部署方案
- `立即部署.md` - 快速部署指南
- `部署指南.md` - 详细部署教程

### 用途：
- 邮箱配置指导
- 部署操作指南
- 快速上手教程

### 🗑️ 何时删除：
完成以下操作后，可以删除整个 `配置指南` 目录：
1. ✅ 已配置好接收建议的邮箱
2. ✅ 已完成项目部署（或不需要部署）
3. ✅ 已熟悉基本操作

## 💡 使用建议

### 开发者：
- 阅读 `/docs/技术文档` 了解项目

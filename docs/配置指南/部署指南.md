# 🚀 圆桌项目部署指南

本指南将帮助你将"圆桌"应用部署到互联网，让任何人都可以通过网址访问。

## 📋 部署前准备

### 1. 构建生产版本

在终端中运行以下命令构建生产版本：

```bash
cd /Users/pinkocean/Desktop/开发/Round\ table/ai-forum
npm run build
```

构建完成后，会在 `dist` 目录生成生产文件。

### 2. 自定义配置（可选）

在部署前，建议修改以下配置：

#### 修改 GitHub 仓库链接
打开 `src/components/SupportModal.jsx`，找到第 186 行：
```javascript
window.open('https://github.com/your-repo', '_blank')
```
替换为你的实际 GitHub 仓库地址。

#### 修改联系邮箱
打开 `src/components/SupportModal.jsx`，找到第 253 行：
```javascript
window.open(`mailto:support@example.com?subject=...`)
```
替换为你的实际联系邮箱。

## 🌐 部署方案（三选一）

### 方案一：Vercel（推荐 ⭐⭐⭐⭐⭐）

**优点**：
- ✅ 完全免费
- ✅ 自动 HTTPS
- ✅ 全球 CDN 加速
- ✅ 自动部署（连接 Git 后）
- ✅ 支持自定义域名

**部署步骤**：

#### 方法 A：使用 Vercel CLI（最简单）

1. **安装 Vercel CLI**
   ```bash
   npm install -g vercel
   ```

2. **登录 Vercel**
   ```bash
   vercel login
   ```
   按照提示使用 GitHub 账号登录。

3. **部署项目**
   ```bash
   cd /Users/pinkocean/Desktop/开发/Round\ table/ai-forum
   vercel
   ```
   
4. **回答提示问题**
   - `Set up and deploy?` → **Y**
   - `Which scope do you want to link to?` → 选择你的账号
   - `Link to existing project?` → **N**
   - `What's your project's name?` → **yuanzhuo**（或自定义）
   - `In which directory is your code located?` → **./**
   - `Want to override the settings?` → **N**

5. **等待部署完成**
   Vercel 会自动构建并部署，完成后会显示访问网址，例如：
   ```
   https://yuanzhuo.vercel.app
   ```

6. **生产环境部署**
   ```bash
   vercel --prod
   ```

#### 方法 B：使用 Vercel 网站

1. 访问 [vercel.com](https://vercel.com)
2. 使用 GitHub 账号登录
3. 点击 "Add New Project"
4. 导入你的 GitHub 仓库
5. 点击 "Deploy"
6. 等待部署完成

### 方案二：Netlify（推荐 ⭐⭐⭐⭐⭐）

**优点**：
- ✅ 完全免费
- ✅ 拖拽部署
- ✅ 自动 HTTPS
- ✅ 支持表单收集

#### 方法 A：使用 Netlify Drop（最简单，无需命令行）

1. **构建项目**
   ```bash
   npm run build
   ```

2. **访问 Netlify Drop**
   打开 [app.netlify.com/drop](https://app.netlify.com/drop)

3. **拖拽部署**
   将 `dist` 文件夹拖拽到网页上的指定区域

4. **获取网址**
   部署完成后会获得一个网址，例如：
   ```
   https://yuanzhuo.netlify.app
   ```

#### 方法 B：使用 Netlify CLI

1. **安装 Netlify CLI**
   ```bash
   npm install -g netlify-cli
   ```

2. **登录 Netlify**
   ```bash
   netlify login
   ```

3. **部署项目**
   ```bash
   cd /Users/pinkocean/Desktop/开发/Round\ table/ai-forum
   netlify deploy
   ```

4. **生产环境部署**
   ```bash
   netlify deploy --prod
   ```

### 方案三：Cloudflare Pages（推荐 ⭐⭐⭐⭐）

**优点**：
- ✅ 完全免费
- ✅ 全球 CDN
- ✅ 无限带宽

**部署步骤**：

1. **访问 Cloudflare Pages**
   打开 [pages.cloudflare.com](https://pages.cloudflare.com)

2. **登录/注册 Cloudflare 账号**

3. **创建项目**
   - 点击 "Create a project"
   - 连接 GitHub 账号
   - 选择你的仓库

4. **配置构建**
   - Framework preset: **Vite**
   - Build command: `npm run build`
   - Build output directory: `dist`

5. **点击 "Deploy"**

## 🔗 自定义域名（可选）

如果你有自己的域名，可以绑定到部署平台：

### Vercel 绑定域名
1. 进入项目设置
2. 点击 "Domains"
3. 添加你的域名
4. 按照提示配置 DNS

### Netlify 绑定域名
1. 进入项目设置
2. 点击 "Domain management"
3. 点击 "Add custom domain"
4. 按照提示配置 DNS

## ⚙️ 环境变量配置（重要）

如果你的应用需要 AI API 密钥，需要在部署平台配置环境变量：

### Vercel 配置环境变量
1. 进入项目设置
2. 点击 "Environment Variables"
3. 添加你的 API 密钥
4. 重新部署

### Netlify 配置环境变量
1. 进入项目设置
2. 点击 "Environment variables"
3. 添加你的 API 密钥
4. 重新部署

# 📧 建议反馈邮箱配置指南

## ✅ 功能说明

现在用户点击"提供建议"按钮后，会自动打开邮件客户端，发送建议邮件到你指定的邮箱。

---

## 🔧 配置步骤

### 步骤 1：打开配置文件

找到并打开文件：
```
/Users/pinkocean/Desktop/开发/Round table/ai-forum/src/config.js
```

### 步骤 2：修改邮箱地址

找到这一行：
```javascript
FEEDBACK_EMAIL: 'your-email@gmail.com', // 请修改为你的实际邮箱地址
```

将 `your-email@gmail.com` 替换为你的实际邮箱地址，例如：
```javascript
FEEDBACK_EMAIL: 'zhangsan@gmail.com',
```

### 步骤 3：修改 GitHub 仓库地址（可选）

如果你想让用户访问你的 GitHub 仓库，修改：
```javascript
GITHUB_REPO: 'https://github.com/your-username/yuanzhuo',
```

替换为你的实际仓库地址，例如：
```javascript
GITHUB_REPO: 'https://github.com/zhangsan/yuanzhuo',
```

### 步骤 4：保存文件

保存 `config.js` 文件后，应用会自动热更新，无需重启服务。

## 📧 支持的邮箱类型

你可以使用任何支持邮件客户端的邮箱：

### 推荐的邮箱服务：
- ✅ **Gmail** - 国际通用，稳定可靠
- ✅ **Outlook/Hotmail** - 微软邮箱
- ✅ **QQ 邮箱** - 国内用户友好
- ✅ **163 邮箱** - 网易邮箱
- ✅ **iCloud Mail** - 苹果邮箱
- ✅ **企业邮箱** - 自定义域名

### 示例：
```javascript
// Gmail
FEEDBACK_EMAIL: 'zhangsan@gmail.com'

// QQ 邮箱
FEEDBACK_EMAIL: '123456@qq.com'

// 163 邮箱
FEEDBACK_EMAIL: 'zhangsan@163.com'

// 企业邮箱
FEEDBACK_EMAIL: 'feedback@yuanzhuo.com'
```

## ⚙️ 技术实现

### 代码逻辑：
```javascript
// 1. 从配置文件读取邮箱
const email = APP_CONFIG.FEEDBACK_EMAIL;

// 2. 生成邮件主题
const subject = encodeURIComponent('圆桌项目建议');

// 3. 生成邮件正文
const body = encodeURIComponent('建议模板内容...');

// 4. 打开邮件客户端
window.open(`mailto:${email}?subject=${subject}&body=${body}`, '_blank');
```

### mailto 协议：
- `mailto:email@example.com` - 指定收件人
- `?subject=主题` - 指定主题
- `&body=内容` - 指定正文
- 自动打开系统默认邮件客户端

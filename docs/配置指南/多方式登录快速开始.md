## 🚀 立即使用

### 本地测试

1. **启动开发服务器**（已运行）
   ```bash
   cd "/Users/pinkocean/Desktop/开发/Round table/ai-forum"
   npm run dev
   ```

2. **打开浏览器**
   - 访问 http://localhost:5175/

3. **点击登录按钮**
   - 点击右上角的"登录"按钮
   - 选择登录方式

4. **测试手机号登录**
   - 选择"手机号登录"
   - 输入任意 11 位手机号
   - 点击"获取验证码"
   - 输入测试验证码：`123456`
   - 点击登录

### 部署到 Netlify 后使用

1. **推送代码**
   ```bash
   git add .
   git commit -m "添加多种登录方式"
   git push origin main
   ```

2. **在 Netlify 启用 Identity**
   - 访问 https://app.netlify.com
   - 选择您的站点
   - 点击 Identity → Enable Identity

3. **配置 OAuth 提供商**（可选）
   - 详见 `第三方登录配置.md`
   - GitHub 和 Google 可立即使用
   - 微信、支付宝、抖音需要额外配置

4. **测试登录**
   - 等待部署完成
   - 打开 Netlify 站点 URL
   - 点击登录测试

## 📝 配置 OAuth（可选）

### GitHub 登录配置（推荐）

1. 访问 https://github.com/settings/developers
2. 创建新的 OAuth App
3. 配置回调 URL: `https://your-site.netlify.app/.netlify/identity`
4. 复制 Client ID 和 Client Secret
5. 在 Netlify Identity 设置中配置

### Google 登录配置（推荐）

1. 访问 https://console.developers.google.com/
2. 创建 OAuth 2.0 客户端 ID
3. 配置回调 URL: `https://your-site.netlify.app/.netlify/identity`
4. 复制 Client ID 和 Client Secret
5. 在 Netlify Identity 设置中配置

# ⚡ 快速设置指南

## 📧 配置建议接收邮箱（必须）

### 1. 打开配置文件
打开文件：`src/config.js`

### 2. 修改邮箱地址
找到这一行：
```javascript
FEEDBACK_EMAIL: 'your-email@gmail.com',
```

替换为你的邮箱，例如：
```javascript
FEEDBACK_EMAIL: 'zhangsan@gmail.com',
```

### 3. （可选）修改 GitHub 地址
```javascript
GITHUB_REPO: 'https://github.com/your-username/yuanzhuo',
```

替换为：
```javascript
GITHUB_REPO: 'https://github.com/zhangsan/yuanzhuo',
```

### 4. 保存文件
按 `Cmd + S` 保存，应用会自动更新。

## ✅ 测试功能

### 测试步骤：
1. 打开应用
2. 点击打赏按钮（❤️）
3. 滚动到"其他支持方式"
4. 点击"提供建议"
5. 检查是否打开邮件客户端
6. 检查收件人是否是你的邮箱

# 第三方登录配置指南

## 支持的登录方式

✅ **已配置（Netlify Identity 原生支持）**
- GitHub
- GitLab
- Bitbucket
- Google
- Facebook

⚠️ **需要额外配置**
- 微信
- 支付宝
- 抖音
- 手机号登录

## 配置步骤

### 1. 在 Netlify 后台启用 OAuth 提供商

#### GitHub 登录配置

1. 访问 https://github.com/settings/developers
2. 点击 "New OAuth App"
3. 填写应用信息：
   - **Application name**: 圆桌
   - **Homepage URL**: 您的 Netlify 站点 URL（如：https://your-site.netlify.app）
   - **Authorization callback URL**: `https://your-site.netlify.app/.netlify/identity`
4. 点击 "Register application"
5. 复制 **Client ID** 和 **Client Secret**
6. 在 Netlify 后台：
   - 进入 Identity → Settings and usage
   - 找到 "External providers"
   - 点击 GitHub → "Install provider"
   - 填入 Client ID 和 Client Secret
   - 点击 Save

#### Google 登录配置

1. 访问 https://console.developers.google.com/
2. 创建新项目或选择现有项目
3. 启用 "Google+ API"
4. 创建 OAuth 2.0 客户端 ID：
   - 应用类型：Web application
   - Authorized JavaScript origins: 您的 Netlify 站点 URL
   - Authorized redirect URIs: `https://your-site.netlify.app/.netlify/identity`
5. 复制 **Client ID** 和 **Client Secret**
6. 在 Netlify 后台配置（同上）

### 2. 配置微信登录（需要自定义）

微信登录需要使用第三方服务或自建 OAuth 服务器：

#### 方案 A：使用 Authing 等身份云服务

1. 访问 https://authing.cn/
2. 注册并创建应用
3. 启用微信登录
4. 配置微信开放平台应用
5. 在 Netlify 中使用 Authing 作为 OAuth 提供商

#### 方案 B：自建 OAuth 服务器

1. 注册微信开放平台开发者
2. 创建网站应用
3. 获取 AppID 和 AppSecret
4. 创建 Netlify Function 处理 OAuth 回调
5. 配置自定义 OAuth 流程

### 3. 配置支付宝登录

1. 访问支付宝开放平台：https://open.alipay.com/
2. 创建应用并申请网站应用
3. 获取 AppID 和 AppSecret
4. 配置回调地址
5. 使用自定义 OAuth 实现

### 4. 配置抖音登录

1. 访问抖音开放平台：https://open.douyin.com/
2. 创建应用
3. 获取 Client Key 和 Client Secret
4. 配置回调地址
5. 使用自定义 OAuth 实现

### 5. 手机号登录配置

当前实现为测试版本，生产环境需要：

#### 方案 A：使用短信服务商

**阿里云短信**
1. 访问 https://dysms.console.aliyun.com/
2. 创建短信签名和模板
3. 获取 AccessKey ID 和 AccessKey Secret
4. 创建 Netlify Function 发送验证码

**腾讯云短信**
1. 访问 https://console.cloud.tencent.com/sms
2. 创建短信签名和模板
3. 获取 SecretId 和 SecretKey
4. 创建 Netlify Function 发送验证码

#### 方案 B：使用 Authing 等身份云服务

1. 在 Authing 中启用手机号登录
2. 配置短信服务商
3. 在应用中使用 Authing SDK

## 测试手机号登录

当前测试版本的手机号登录：

- 输入任何 11 位手机号 + 123456 即可登录

## 环境变量配置

在 Netlify 后台配置环境变量：

1. 进入 Site settings → Build & deploy → Environment
2. 添加以下变量：

```bash
# GitHub OAuth
GITHUB_CLIENT_ID=your_github_client_id
GITHUB_CLIENT_SECRET=your_github_client_secret

# Google OAuth
GOOGLE_CLIENT_ID=your_google_client_id
GOOGLE_CLIENT_SECRET=your_google_client_secret

# 微信 OAuth (如使用)
WECHAT_APP_ID=your_wechat_app_id
WECHAT_APP_SECRET=your_wechat_app_secret

# 阿里云短信 (如使用)
ALIYUN_ACCESS_KEY_ID=your_access_key_id
ALIYUN_ACCESS_KEY_SECRET=your_access_key_secret
ALIYUN_SMS_SIGN_NAME=your_sms_sign_name
ALIYUN_SMS_TEMPLATE_CODE=your_sms_template_code
```

## 常见问题

### Q: 第三方登录回调失败？
A: 确保回调 URL 配置正确，格式为：`https://your-domain.com/.netlify/identity`

### Q: 微信/支付宝/抖音登录无法使用？
A: 这些需要自定义 OAuth 实现，建议使用 Authing 等第三方服务简化配置

### Q: 手机号登录收不到验证码？
A: 当前为测试版本，使用固定验证码 123456。生产环境需要配置短信服务商

### Q: 如何只启用部分登录方式？
A: 在 `netlify.toml` 中配置 `[identity.external_providers]`，设置为 `false` 即可禁用

# 🚀 永久部署方案

## 问题说明
Netlify Drop 临时部署的网址只能存在 1 小时且有密码保护。

## 解决方案：使用正式账号部署（永久、公开、免费）

---

## 🎯 推荐方案：Vercel 正式部署

### 步骤 1：访问 Vercel 官网
打开浏览器访问：
```
https://vercel.com
```

### 步骤 2：注册/登录账号
- 点击 "Sign Up" 或 "Login"
- **使用 GitHub 账号登录**（推荐）
- 或者使用邮箱注册

### 步骤 3：创建新项目
1. 登录后点击 "Add New Project"
2. 点击 "Import Git Repository"
3. 选择你的 GitHub 账号授权

### 步骤 4：导入项目
有两种方式：

#### 方式 A：已有 GitHub 仓库
- 选择你的 "Round table" 仓库
- 点击 "Import"

#### 方式 B：创建新仓库
1. 先在 GitHub 创建新仓库
2. 将代码 push 到仓库
3. 然后在 Vercel 导入

### 步骤 5：配置项目
- **Framework Preset**: Vite
- **Build Command**: `npm run build`
- **Output Directory**: `dist`
- **Install Command**: `npm install`

### 步骤 6：点击 Deploy
等待 1-2 分钟，部署完成后你会获得：
```
https://yuanzhuo.vercel.app
```

## 🎯 备选方案：Netlify 正式部署

### 步骤 1：访问 Netlify 官网
```
https://app.netlify.com
```

### 步骤 2：注册/登录
- 使用 GitHub 账号登录（推荐）
- 或邮箱注册

### 步骤 3：创建项目
1. 点击 "Add new site"
2. 选择 "Import an existing project"
3. 连接 GitHub

### 步骤 4：选择仓库
- 选择你的 "Round table" 仓库

### 步骤 5：配置构建
- **Build command**: `npm run build`
- **Publish directory**: `dist`

### 步骤 6：点击 Deploy
等待完成后获得：
```
https://yuanzhuo.netlify.app
```

## 📝 如果你还没有 GitHub 仓库

### 创建 GitHub 仓库步骤：

1. **访问 GitHub**
   ```
   https://github.com
   ```

2. **登录/注册 GitHub 账号**

3. **创建新仓库**
   - 点击右上角 "+" → "New repository"
   - Repository name: `yuanzhuo` 或 `round-table`
   - 选择 **Public**（公开）
   - 不要初始化（不要勾选 README）
   - 点击 "Create repository"

4. **上传代码**
   
   在终端执行：
   ```bash
   cd /Users/pinkocean/Desktop/开发/Round\ table/ai-forum
   
   # 初始化 git
   git init
   
   # 添加所有文件
   git add .
   
   # 提交
   git commit -m "Initial commit"
   
   # 添加远程仓库（替换 YOUR_USERNAME 为你的 GitHub 用户名）
   git remote add origin https://github.com/YOUR_USERNAME/yuanzhuo.git
   
   # 推送
   git branch -M main
   git push -u origin main
   ```

## ✅ 部署后的优势

### Vercel/Netlify 正式部署：
- ✅ **永久存在**（只要账号存在）

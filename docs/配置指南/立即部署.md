## 🚀 三种快速部署方式（选择一种即可）

### 方式一：使用 Vercel CLI（推荐，最快）

**执行步骤：**

1. **打开终端**，运行：
   ```bash
   cd /Users/pinkocean/Desktop/开发/Round\ table/ai-forum
   vercel --prod
   ```

2. **按提示操作**：
   - 如果未登录：运行 `vercel login` 登录
   - 第一次部署时：
     - `Link to existing project?` → 输入 **N**
     - `Project name?` → 输入 **yuanzhuo**（或自定义）
     - `In which directory is your code located?` → 直接回车（默认 ./）
     - `Want to override the settings?` → 输入 **N**

3. **等待部署完成**（约 30 秒）

4. **获取网址**：
   部署完成后会显示：
   ```
   https://yuanzhuo-xxx.vercel.app
   ```

---

### 方式二：使用 Netlify Drop（最简单，无需命令）

**执行步骤：**

1. **打开浏览器**访问：
   ```
   https://app.netlify.com/drop
   ```

2. **拖拽部署**：
   - 打开访达（Finder）
   - 找到 `dist` 文件夹（在 `/Users/pinkocean/Desktop/开发/Round table/ai-forum/dist`）
   - 将 `dist` 文件夹拖拽到网页上的虚线框内

3. **等待上传完成**（约 10-30 秒）

4. **获取网址**：
   部署完成后会显示：
   ```
   https://xxx-xxx.netlify.app
   ```

5. **（可选）注册账号**：
   可以注册 Netlify 账号来管理你的项目

---

### 方式三：使用 Vercel 网站

**执行步骤：**

1. **访问 Vercel**：
   ```
   https://vercel.com
   ```

2. **登录**：
   使用 GitHub 账号登录

3. **导入项目**：
   - 点击 "Add New Project"
   - 选择 "Import Git Repository"
   - 选择你的 GitHub 仓库
   - 点击 "Deploy"

4. **等待部署完成**

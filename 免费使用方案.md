## 方案二：完全免费的开源方案

### 1. Hugging Face Inference API
- **价格**：完全免费（有速率限制）
- **官网**：https://huggingface.co
- **模型**：数千个开源模型
- **限制**：每分钟调用次数有限

**配置方法**：
1. 注册 Hugging Face 账号
2. 获取 Access Token
3. 在设置中选择 Hugging Face 服务商

### 2. Ollama（本地运行，完全免费）
- **价格**：完全免费
- **官网**：https://ollama.ai
- **优点**：本地运行，无需 API，隐私安全
- **缺点**：需要自己的电脑性能较好

**配置方法**：
```bash
# 安装 Ollama
brew install ollama  # macOS
# 或
curl -fsSL https://ollama.ai/install.sh | sh  # Linux

# 下载模型
ollama pull llama2
ollama pull qwen2.5

# 启动服务
ollama serve
```

然后在圆桌中配置：
- API 地址：http://localhost:11434
- 模型：llama2 或 qwen2.5

### 3. LM Studio（本地运行，图形界面）
- **价格**：完全免费
- **官网**：https://lmstudio.ai
- **优点**：图形界面，简单易用
- **支持**：Windows/macOS/Linux

**配置方法**：
1. 下载并安装 LM Studio
2. 下载喜欢的模型
3. 启动本地服务器
4. 配置 API 地址：http://localhost:1234

### 4. Groq（速度快，有免费额度）
- **价格**：免费额度充足
- **官网**：https://groq.com
- **优点**：速度极快
- **模型**：Llama、Mixtral 等

## ⚠️ 注意事项

1. **免费额度的限制**
   - 大多数免费额度有有效期（如 30 天）
   - 注意查看过期时间

2. **速率限制**
   - 免费 API 通常有调用频率限制
   - 建议设置合理的请求间隔

3. **模型质量**
   - 免费模型可能不如付费模型效果好
   - 重要场景建议使用付费服务

4. **隐私安全**
   - 本地部署（Ollama/LM Studio）最安全
   - 云服务注意不要上传敏感信息

## 🔧 使用方法

### 在组件中使用翻译

```jsx
import { useForumStore } from '../store';
import { t } from '../i18n';

function MyComponent() {
  const { language } = useForumStore();
  
  return (
    <div>
      <h1>{t('settings.title', language)}</h1>
      <button>{t('action.save', language)}</button>
      <input placeholder={t('topic.placeholder.title', language)} />
    </div>
  );
}
```

### 带参数的翻译

```jsx
// 使用参数
t('character.selectedCount', language, { count: activeCharacters.length })
// 输出：已选择 3 个角色 / 3 characters selected / 3 キャラクター選択済み
```
